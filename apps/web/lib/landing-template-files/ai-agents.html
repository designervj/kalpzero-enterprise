FAQ

Answers You Need

Everything you need to know before your first agent goes live.

How quickly can I deploy my first AI agent?

+

Most users deploy their first working agent within 15 minutes of signing up. Our no-code builder lets you describe your agent in plain language, connect your tools, and ship — no engineering required. For complex multi-agent systems, our quickstart templates get you to production in under an hour.

Which AI models power the agents?

+

Our platform is model-agnostic by design. You can use OpenAI GPT-4o, Anthropic Claude, Google Gemini, Mistral, Llama 3, and over a dozen others. Our smart routing layer can automatically select the best model per task based on cost, speed, and capability requirements.

Is my data used to train your models?

+

Absolutely not. We operate under a strict zero-training-data policy. Your agent configurations, task inputs, outputs, and conversation logs are never used to train or fine-tune any AI models — ours or third-party providers'. All data is encrypted at rest and in transit.

Can I run agents on my own infrastructure?

+

Yes — our Enterprise plan includes on-premise and private cloud deployment options. You get the full AI Agents platform running within your own VPC or data center, with complete isolation from our shared infrastructure and full control over your data sovereignty.

How do I connect my existing tools and APIs?

+

We offer 180+ pre-built integrations for Salesforce, Notion, Slack, GitHub, PostgreSQL, Stripe, and more. For custom tools, you can register any REST or GraphQL API in minutes using our Tool Registry. We also support MCP (Model Context Protocol) for standardized tool calling.

What happens if an agent fails mid-task?

+

Our execution engine includes automatic error detection, contextual retry strategies, and graceful fallback behaviors. Every failed task is captured in full with a complete trace for debugging. You can configure alert rules, automatic escalation to human-in-the-loop review, and custom recovery workflows.

Developer API

Build on the API

A clean, RESTful API with real-time streaming, webhooks, and SDKs for Python, TypeScript, and Go. Integrate AI Agents into any stack in minutes.

POST/v1/agentsCreate agent
POST/v1/agents/{id}/runExecute task
GET/v1/runs/{id}Get run status
GET/v1/runs/{id}/traceInspect trace
DEL/v1/agents/{id}Delete agent
